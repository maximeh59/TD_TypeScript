# TD TypeScript NodeJS - Création d'une API

## Description

Création d'une **API** développée avec **Node.js** et le **framework Express**. Il sert de base pour comprendre et implémenter les opérations **CRUD** (Create, Read, Update, Delete) essentielles sur une ressource (dans ce cas, des utilisateurs) et illustre la gestion des requêtes HTTP (GET, POST, PUT, DELETE).
Toutefois, ici uniquement les requêtes GET et POST sont gérées.

---

## Technologies Utilisées

* **Node.js** : Environnement d'exécution JavaScript côté serveur.
* **Express** : Framework web pour créer l'API.
* **Dotenv** : Gestion des variables d'environnement.
* **TypeScript** : Langage de programmation pour le scripting.
* **ts-node** : Exécute le TypeScript sans compilation explicite.
* **Nodemon** : Recharge le serveur en cas de modification.

---

## Démarrage Rapide

### Prérequis

Assurez-vous d'avoir Node.js et npm (Node Package Manager) installés.

### Installation

1.  Clonez ce dépôt :
    ```bash
    git clone https://github.com/maximeh59/TD_TypeScript
    cd TD_TypeScript
    ```
2.  Installez les dépendances :
    ```bash
    npm install
    ```

### Lancement du Serveur

Lancez l'application :

```bash
npm run dev
```

Port du serveur : 4000
